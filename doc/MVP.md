# MVP 

## Описание

Для жизни способности проекта нужно продумать как в игровой форме представить решение примеров.

В MVP API будут следующие моменты:
- Система последовательности математических концепций до 7 класса;
- Система оценки уровня знаний;
- Движок генерации примеров.

## Целевая аудитория

Школьники до 7 класса точнее их родители.

Родителей нужно заинтересовать в полезности проекта для их детей.

Школьников нужно заинтересовать в решение математических задач.

## Сущности
В MVP API будут следующие сущности:
- Complexity - уровень сложности
- Expression - конкретная задача\пример для решения.
- Solution - решение конкретной задачи.
