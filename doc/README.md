# Проект math face 🤓

## Краткое описание

Платформа изучения математики. Проект помогает понять математические концепции. 

В проекте все концепции разбиты на уровни сложности и упорядочены. 

Общая последовательность действий:
1. Оценка уровня знаний;
2. Определений целей(log, sin, кв. уравнения и т.д.)
3. Выполнение заданий с постепенным повышением сложности.



## Миссия проекта

Доказать, что математика - это просто 😎.
