## Описание сущностей

Complexity - уровень сложности
- level - цифровое представление сложности, где операции сложения\вычитания = 1 и тд.
- template - шаблон примера("\d + \d")
- description - описание уровня сложности
- max_value - максимальное значение для операндов. Выход за них может означать повышение сложности

Expression - конкретная задача\пример для решения. 
- complexity_id - id сложности выражения примера
- solution_id - id решения
- value - конкретный пример с решением("2 + 4")
- create_time - время создания
- solution_time - время решения

Solution - решение конкретной задачи.
- value - решение("6)
- description - описание решения
